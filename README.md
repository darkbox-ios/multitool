## Installation

1. Get the App URL by placing "/www/index.html/" at the end of the URL
2. Click the share icon > Add to Homescreen
3. Open the PWA

### Prerequisites

- A Browser
- Mobileprovision/p12/ipas or whatever you need for the utility inside the app.
